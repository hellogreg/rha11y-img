Getting the bookmarklet

To get the rha11y-meta-og bookmarklet, drag the following link to your bookmarks bar. You may also click the link directly to use it on this page for testing. (Refresh the page to reset it.)

rha11y-meta-og

The rha11y-meta-og bookmarklet gathers all Open Graph meta tag information for the current page. When run, the script opens a modal that displays these tags' properties and values.
