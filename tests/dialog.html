<dialog> modal tests

The W3c's ARIA Authoring Practices Guide states that "Windows under a modal dialog [should be] inert. That is, users cannot interact with content outside an active dialog window."

However, it is possible to have some interaction with the underlying content when a <dialog> modal is opened. Mouse users can click outside the modal and scroll up and down the page. And both mouse and keyboard users can follow links within the modal that point to in-page anchors on the underlying page.

Activate dialog
